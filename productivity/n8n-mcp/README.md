# n8n MCP Server

An MCP server that allows Claude Desktop, Claude Code, and other editors to inspect, create, and build workflow automations inside n8n.

> Part of **[MCP Bazaar](../../README.md)** · [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)

- **Source URL**: [https://github.com/czlonkowski/n8n-mcp](https://github.com/czlonkowski/n8n-mcp)

- **License**: MIT
- **Transport**: `stdio`
- **Date Added**: 2026-06-20

## Tools Exposed
- `list_workflows`:  Retrieve a list of n8n workflows.
- `get_workflow`:  Fetch a specific workflow JSON configuration.
- `create_workflow`:  Construct or modify a workflow.
- `execute_workflow`:  Trigger a workflow run.
