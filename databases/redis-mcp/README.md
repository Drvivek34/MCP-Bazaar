# Redis MCP Server

A Model Context Protocol server providing key-value operations, lists, sets, and pub/sub commands on Redis instances.

## Tools Exposed
- `redis_get`:  Retrieve value of key.
- `redis_set`:  Write value of key.
- `redis_keys`:  Scan keys matching pattern.
