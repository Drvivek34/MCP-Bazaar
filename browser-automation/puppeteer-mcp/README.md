# Puppeteer Web Browser MCP

An MCP server that orchestrates headless Chrome browsers using Puppeteer, letting models click, screenshot, and parse live web pages.

> Part of **[MCP Bazaar](../../README.md)** · [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)

## Details
- **Source URL**: [https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer](https://github.com/modelcontextprotocol/servers/tree/main/src/puppeteer)
- **Author**: Model Context Protocol Team
- **License**: MIT
- **Transport**: `stdio`
- **Date Added**: 2026-06-21

## Tools Exposed
- `browser_navigate`:  Move to a specific URL.
- `browser_click`:  Trigger click event on selector.
- `browser_screenshot`:  Capture screenshot png file.
