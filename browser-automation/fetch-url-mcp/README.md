# Fetch HTTP URL MCP Server

Downloads web pages, parses raw HTML, and returns readable markdown. Perfect for static documentation crawls.

> Part of **[MCP Bazaar](../../README.md)** · [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)

## Tools Exposed
- `fetch_content`:  Download and parse URL to markdown.
- `fetch_headers`:  Retrieve response headers from endpoint.
