# GitHub MCP Server

GitHub's official Model Context Protocol server enabling agents to read/write repositories, manage issues, pull requests, and review code.

> Part of **[MCP Bazaar](../../README.md)** · [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)

## Details
- **Source URL**: [https://github.com/github/github-mcp-server](https://github.com/github/github-mcp-server)
- **Author**: GitHub
- **License**: MIT
- **Transport**: `stdio`
- **Date Added**: 2026-06-20

## Tools Exposed
- `create_or_update_file`:  Create or update a file in a repository.
- `search_code`:  Search for code inside a repository.
- `get_issue`:  Retrieve issue details.
- `create_pull_request`:  Open a new pull request.
