# Slack MCP Server

Slack workspace connector allowing AI models to read channels, post messages, search history, and list users.

> Part of **[MCP Bazaar](../../README.md)** · [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)

## Details
- **Source URL**: [https://github.com/modelcontextprotocol/servers/tree/main/src/slack](https://github.com/modelcontextprotocol/servers/tree/main/src/slack)
- **Author**: Model Context Protocol Team
- **License**: MIT
- **Transport**: `stdio`
- **Date Added**: 2026-06-21

## Tools Exposed
- `post_message`:  Send a message to a Slack channel.
- `get_channel_history`:  Read recent messages from a channel.
- `search_messages`:  Query channel histories for keywords.
