# Memory & Knowledge Base MCP

Graph-based semantic memory server allowing models to store facts, entities, and relationships persistently across chats.

> Part of **[MCP Bazaar](../../README.md)** · [Mega AI Bazaar](https://drvivek34.github.io/Mega-AI-Bazaar/)

- **Date Added**: 2026-06-21

## Tools Exposed
- `add_fact`:  Insert a fact entity into memory graph.
- `query_entities`:  Find facts linked to entity name.
- `delete_fact`:  Remove a stale fact from memory.
