# 🔌 MCP Bazaar

> A categorized collection of Model Context Protocol (MCP) servers and tools.

[![Mega AI Bazaar](https://img.shields.io/badge/🌐_Mega_AI_Bazaar-browse_all-6C5CE7)](https://drvivek34.github.io/Mega-AI-Bazaar/) [![Submit](https://img.shields.io/badge/➕_Submit-a_MCP_servers-2ecc71)](https://github.com/Drvivek34/MCP-Bazaar/issues/new/choose) [![License](https://img.shields.io/badge/License-MIT-blue)](LICENSE)

MCP Bazaar collects **Model Context Protocol servers** — the plug-in tools and resources that connect LLMs to databases, APIs, browsers, and more — into one categorized catalog. Top servers are surfaced in **featured/**.

## 📂 Categories

**[⭐ Featured](featured/)** — top picks, surfaced at the very front.

| Category | About |
|---|---|
| [AI & ML](ai-ml/) | Model, embedding, and inference servers. |
| [Browser Automation](browser-automation/) | Playwright, Puppeteer, and headless-browser servers. |
| [Cloud Services](cloud-services/) | AWS, GCP, Azure, and cloud platform servers. |
| [Communication](communication/) | Slack, email, Discord, Telegram, and messaging servers. |
| [Databases](databases/) | MCP servers for SQL/NoSQL, vector DBs, and warehouses. |
| [Developer Tools](dev-tools/) | Linters, runners, debuggers, and toolchain servers. |
| [E-commerce](ecommerce/) | Stores, catalogs, and order servers. |
| [Filesystems & Storage](filesystems-storage/) | Local FS, object storage, and file management servers. |
| [Finance](finance/) | Markets, payments, accounting, and crypto servers. |
| [IoT & Hardware](iot-hardware/) | Device, sensor, and hardware-control servers. |
| [Knowledge & Memory](knowledge-memory/) | RAG, memory, and knowledge-graph servers. |
| [Maps & Location](maps-location/) | Geocoding, routing, and location servers. |
| [Media](media/) | Image, audio, and video servers. |
| [Miscellaneous](misc/) | Everything that doesn't fit a category yet. |

## 🗂️ How it's organized
Every MCP servers lives in its **own sub-folder** inside a category, with a `README.md` recording its **source, author, and original license** (a full copy of the files is kept). See **[CONTRIBUTING.md](CONTRIBUTING.md)**.
